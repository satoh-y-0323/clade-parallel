# /agent-tester コマンド

テストエージェント（tester）をサブエージェントとして起動する。

## ルールの読み込み
**起動時の最初のアクションとして** `.claude/skills/agents/tester.md` を Read し、ルールを確認してから作業を開始すること。

## 動作
Agent ツールで `subagent_type: tester` を指定して起動する。
現在の作業コンテキスト（ユーザーの依頼内容・既存レポートの有無）をプロンプトに含めること。

## 用途
- テスト仕様設計・失敗テスト作成（TDD の Red フェーズ）
- developer の実装後のテスト実行・結果確認
- テストレポート（`test-report-*.md`）の作成

## 注意
- ソースファイルの編集・書き込みは行わない
- 「おそらく動く」という推測での PASS は禁止（必ず実行して確認する）
